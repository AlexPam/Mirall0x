### Installation

On windows using venv:
```bash	
git clone https://github.com/OpenDataforWeb3/Mirall0x
cd Mirall0x
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
streamlit run main\\main\\mirall0x.py
```
